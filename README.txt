STARTING NODE.JS

1. install node.js (https://nodejs.org/en) your file must be installed in C:\ Folder
2. open cmd (run as admin)
3. write these commands each individually:
	cd your\path\to\PRINCEWEBADMIN (change "your\path\to\" to your actual path)
	mkdir email-sender
	cd email-sender
	npm init -y
	npm install express nodemailer cors
4. run the node.js server:
	node sendEmail.js

---------------------------------------------------------------------------------------------------------------
NOTE: IF YOU ALREADY INSTALLED AND CONFIGURED ALL IN NODE.JS,JUST RESUME TO STEP 4 (run the node.js server) 
EVERYTIME YOU OPEN THE SYSTEM SO THAT THE EMAIL NOTIFICATION WILL WORK.
